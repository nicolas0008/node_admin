## Installation

```bash
$ npm install
```

## Running the app

```bash
# prestart command
$ npm run prestart

# development
$ npm run start

# watch mode
$ npm run start:debug

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
